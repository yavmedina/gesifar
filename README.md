# GESIFAR - Sistema de Gestión de Farmacia Hospitalaria

Sistema integral de gestión para farmacias hospitalarias desarrollado en PHP y MySQL.

## 🎯 Objetivo

Proyecto final para la carrera de Analista de Sistemas - Gestión de inventario y dispensación de medicamentos en entorno hospitalario.

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Base de datos**: MySQL
- **Servidor**: Apache (Raspberry)/Laragon (para pruebas de implementaciones)

## 📋 Funcionalidades Principales

- Gestión de inventario de medicamentos
- Control de stock y vencimientos
- Registro de recetas médicas
- Dispensación de medicamentos
- Reportes y estadísticas
- Sistema de usuarios y permisos
